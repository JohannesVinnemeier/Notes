\section{Chern Weil Theory}

In this section we review the construction of the Chern-Weil homomorphism. This is largely based on \cite{emilia_globale_ana} but we add the missing details, fix some mistakes and give more precise definitions. In the following let $\mathbb{K} \in \{\R, \C\}$. All manifolds will be real but we consider real was well as complex vector bundles over them. If $\mathbb{K} =  \R$ the vector bundles are understood to be real and if $\mathbb{K} =  \R$ they are understood to be complex. There is virtually no difference between the complex and the real case in what follows.
